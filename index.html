プログラム復習コンテンツ

配列-4

サンプルコードの結果

サンプルコード

　　 float[] gray;

　　 　

　　 void setup(){

　　 　size(240, 120);

　　 　gray=new float[width];

　　 　for(int i=0; i　　 　　gray[i]=random(0, 255);

　　 　}

　　 }

　　 　

　　 void draw(){

　　 　for(int i=0; i　　 　　stroke(gray[i]);

　　 　　line(i, 0, i, height);

　　 　}

　　 }

説明文

　配列-4では、forループを使用した配列のプログラムである。

　

　このプログラムでは、setup()で乱数を代入し、その値を使用しdraw()で線を描いている。

　プログラムが実行されるたびに新しい乱数のセットが配列に入る。

　プログラム中にある[.length]というコードは、配列の数を出力する際に用いる。
